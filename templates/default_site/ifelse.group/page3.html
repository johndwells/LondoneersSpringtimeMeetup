    limit="3"
    status="{if segment_3 == "featured"}featured{/if}"}
    {title} - {status}<br>
{/exp:channel:entries}

What happens when this is run? {if segment_3 == ""} toggle {if:else} toggle {/if} 

{exp:channel:entries limit="3" status="{if segment_3 == "featured"}featured{/if}"}{title} - {status}
{/exp:channel:entries}

Demo: Advanced Conditionals as Tag Parameters

{exp:channel:entries
	limit="3"
	status="{if get:status}{get:status}{if:else}open{/if}"}
	{title} - {status}<br>
{/exp:channel:entries}

What happens when this is run? {if get:status == "featured"} toggle {if:else} toggle {/if} 

{exp:channel:entries limit="3" status="{if get:status}{get:status}{if:else}open{/if}"}
{title} - {status}{/exp:channel:entries}

That's right, nothing. Because Channel Entries is looking for a status that matches "(if get:status)(get:status)(if:else)open(/if)".
